# Security Policy

## Reporting a vulnerability

If you find a security issue in Video Tuner Pro, please report it privately
instead of opening a public issue:

- Email **contact@slonick.dev**, or
- Use GitHub's [private vulnerability reporting](https://github.com/Slonick/Video-Tuner-Pro/security/advisories/new).

Please include steps to reproduce, the affected version, and the browser you
saw it on. I'll acknowledge your report as soon as I can and keep you updated
on the fix.

## Supported versions

Only the latest published version receives security fixes.
