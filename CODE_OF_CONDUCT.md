# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1. Please read it — it defines the standards we expect and the
behavior that is not tolerated.

In short: be respectful and constructive. Everyone is welcome to participate
regardless of background or identity, and the project should stay a friendly,
harassment-free space for all.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the
maintainer at **contact@slonick.dev**. Reports are handled confidentially and
reviewed as promptly as possible.
